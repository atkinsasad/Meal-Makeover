# 🌿 Meal Makeover

A Claude-powered web app that helps people with diabetes and prediabetes enjoy the foods they love with less impact on their blood sugar.

## The Problem
Over 130 million Americans live with diabetes or prediabetes. One of the biggest daily challenges is navigating food — not knowing which meals are safe, or feeling like healthy eating means giving up the dishes they enjoy.

## The Solution
Meal Makeover lets users enter any meal idea and instantly receive a personalized, blood-sugar-friendly alternative. The app suggests specific ingredient swaps (e.g. cauliflower mash instead of mashed potatoes), respects dietary restrictions like gluten-free or vegan, and — when no swap exists — recommends the clinically supported "fiber first" eating order to reduce glucose spikes without changing the meal at all.

## Built With
- [Claude Sonnet 4.6](https://anthropic.com) via the Anthropic API
- React

## How to Run
1. Clone this repo
2. Drop `meal_makeover.jsx` into a React project (e.g. [StackBlitz](https://stackblitz.com/new))
3. The app calls the Anthropic API directly — no backend needed
